# Code of Conduct

`vietnam-address-kit` is an open-source project for developers working with Vietnam address data. We want the community to be respectful, practical, and welcoming.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while keeping technical discussions evidence-based.
- Give actionable feedback instead of personal criticism.
- Respect different experience levels and backgrounds.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic disruption.
- Misrepresenting sample or unverified data as official data.

## Reporting

Report conduct issues through GitHub issues if public discussion is appropriate, or contact the maintainers privately when sensitive information is involved.

Maintainers may remove comments, close discussions, or restrict participation when necessary to protect the community.
